# System Utilities Collection

A collection of professional system utilities and scripts for Linux environment management.

## Contents

1. **binary_search.c** - Binary search algorithm implementation in C
2. **create_user.sh** - Automated user and group management script
3. **capslock_control.c** - CapsLock LED control utility
4. **welcome_message.sh** - System welcome message with information
5. **capslock_toggle.c** - Interactive CapsLock toggle utility
6. **setup_environment.sh** - Shell environment configuration script

## Building and Usage

See individual file headers for compilation and usage instructions.

## Requirements

- Linux system
- GCC compiler for C programs
- Bash shell for scripts
- Root privileges for system operations
